# Mean-Reverting-Stochastic-Volatility

This repository contains the Python code developed for a project on the construction and validation of a numerical framework for a Mean Reverting Stochastic Volatility model.

The work focuses on implementing the model through Monte Carlo simulation and on reproducing the numerical results required by the assignment.

## Repository Contents

- `ASSIGNMENT.pdf`  
  Contains the full statement of the project, including the model specifications and all the requested tasks.

- `SOLUTION.pdf`  
  Contains the formal answers to the questions stated in `ASSIGNMENT.pdf`.

- `PROJECT.py`  
  Python script containing all the functions developed for the Monte Carlo simulation and for the implementation of the Mean Reverting Stochastic Volatility model.

## Structure of the Python Code

The Python script is organized into sections:

- The first sections define the functions used to simulate the specified model.
- The last two sections apply the model directly to satisfy the requests presented in `ASSIGNMENT.pdf`.

This structure makes the script useful both as a reusable implementation of the model and as a complete reproduction of the project results.

## How to Use the Code

If you want to reproduce the full project, follow these steps:

1. Read `ASSIGNMENT.pdf` to understand the model specifications and the required tasks.
2. Run the Python script `PROJECT.py`.
3. Use the first sections of the script if you only need the model functions.
4. Use the last two sections if you want to reproduce the numerical results and the answers to the assignment.
