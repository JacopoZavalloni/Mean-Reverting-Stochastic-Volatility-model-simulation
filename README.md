The work constist in the construction and validation of a Numerical framework for a Mean Reverting Stochastic Volatility model. The python script contain all functions I have constructed for the Monte Carlo simulation.
In the ASSIGNEMENT.pdf file you can read all the specification on the model implemented and all the requests ansered by my job.
You can download the python script in which are implemented the functions; the first sections involve the functions constructed to simulate the specified model while last two sections are the application of the model itself to satisfy the requests seen in the ASSIGNMENT.pdf file. Moreover, the SOLUTION.pdf file contain formal answer to the requests in the ASSIGNEMENT.pdf file.
